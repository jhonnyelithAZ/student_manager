# student_manager

Console program for managing the student sector, using functions, lists, and data entry,
belonging to the riwi performance test.

It includes eight interactive options, which are:

- register student
- delete student
- update student
- show a specific student
- show the entire list of registered students
- save the list and load it when needed



It also allows you to return to the main menu if you make a mistake typing an option or entering a name.

The language used is Python

## Lift locally

Step one:

- Clone the repository (git clone)


Step two:

- python3 main.py
